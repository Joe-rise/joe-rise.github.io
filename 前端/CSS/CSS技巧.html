# 权重

内联样式（1000） > id 选择器（100） > class 选择器（10） > 标签选择器（1）。括号内分别是它们的权重值！

# 居中技巧

  1. mx-auto 左右边距为自动

# width

  1. 设置div max-width: 600px;小屏幕下可以自动缩小
  2. 当你设置一个元素为 box-sizing: border-box; 时，此元素的内边距和边框不再会增加它的宽度

# position 布局

position: relative  相对上一个元素偏移
position: fixed  相对视窗来定位，这意味着即便页面滚动，它还是会停留在相同的位置
absolute absolute 与 fixed 的表现类似，但是它不是相对于视窗而是相对于最近的“positioned”祖先元素。

# Div

改为行内布局 display: inline-block

# Margin

a. Margin:20px  上下左右为20
b. margin：25px 50px 上下为25，左右为50
c. margin：25px 50px  75px 上25，左右50，下75

# 边框，阴影

a. 阴影 box-shadow: 0 2px 12px 0 rgba(0,0,0,0.1)
b. 边框  border-radius: 8px; border: 1px solid rgba(0, 0, 0, 0.1)
边框简写 border: 2px solid red

# Flex 布局

# Justify-content 项目在主轴的对齐方式

flex-start 默认值，左对齐
flex-end 右对齐
center 居中
space-between 两端对齐，项目之间的间隔都相
space-around 类似space-between和两边有间距

# Align-items 项目在交叉轴上如何对齐

flex-start 交叉轴的起点对齐
flex-end 交叉轴的终点对齐
center 交叉轴的中点对齐
baseline 项目的第一行文字的基线对齐

# Grid 布局

# 属性

  • minmax() 函数产生一个长度范围，表示长度就在这个范围之中。它接受两个参数，分别为最小值和最大值。
  • auto 关键字 表示由浏览器自己决定长度

# 三行三列的网格

.container {
display: grid;
grid-template-columns: 100px 100px 100px;
grid-template-rows: 100px 100px 100px;
}

# 每列宽度为100px，根据容器宽度自动填充列

.container {
display: grid;
grid-template-columns: repeat(auto-fill, 100px);
}
表示每列宽度100px，然后自动填充，直到容器不能放置更多的列

# 宽度相同的列

.container {
display: grid;
grid-template-columns: 1fr 1fr;
}
